## Lenguaje Javascript
JavaScript (JS) es un lenguaje de programación fundamental para la interactividad web, permitiendo animaciones, validación de formularios y contenido dinámico, trabajando junto a HTML y CSS para crear experiencias ricas; es versátil, multiparadigma y se ejecuta en el navegador, siendo la base de frameworks populares como React, Angular y Vue.js, indispensable para el desarrollo web moderno y más allá. 
```
const helloWorld = "Hello World" 
console.log(helloWorld)
```
